Sobre o Projeto

Este site foi desenvolvido como parte de uma atividade escolar com o objetivo de apresentar conteúdos históricos de forma organizada, acessível e interessante. A proposta é reunir informações sobre acontecimentos, períodos e personagens importantes da História, ajudando no aprendizado e na divulgação do conhecimento histórico.

O projeto foi desenvolvido pela equipe formada por Rafael Celio, Pedro Lucas e Max Matheus, utilizando as tecnologias HTML, CSS e JavaScript para criar uma experiência simples, funcional e agradável.
